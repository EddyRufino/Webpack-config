# Webpack Config

Proyecto base para comenzar a desarrollar aplicaciones utilizando Webpack.

### Ejecutar:
Reconstruir el modulo de node:
```
npm install
```
Para ejecutar el proyecto:
```
npm start
```

### Notas:
